## Resume Customization Guidelines by Company Tier

### FAANG Customization Strategy

**Focus Areas**: Scale, performance, innovation
**Key Modifications**:

- Emphasize user numbers (10K+, 100K+, 1M+)
- Highlight performance metrics (latency, throughput)
- Include innovation aspects (novel approaches, research)
- Stress system reliability (99.9% uptime)

### Unicorn Customization Strategy

**Focus Areas**: Growth, scalability, modern tech stack
**Key Modifications**:

- Emphasize growth metrics and scalability
- Highlight modern technology adoption
- Include rapid iteration and deployment
- Stress user engagement and retention

### Established Tech Customization Strategy

**Focus Areas**: Enterprise features, reliability, integration
**Key Modifications**:

- Emphasize enterprise-grade features
- Highlight system integration capabilities
- Include security and compliance aspects
- Stress reliability and stability

### Startup Customization Strategy

**Focus Areas**: Versatility, rapid development, innovation
**Key Modifications**:

- Emphasize full-stack versatility
- Highlight rapid prototyping and iteration
- Include entrepreneurial experience (Inexia)
- Stress adaptability and learning agility

## Research Sources and Company Intelligence

### Primary Research Sources

1. **Company Engineering Blogs**: Technical challenges, tech stack, culture
2. **GitHub Organizations**: Open source projects, coding standards
3. **LinkedIn Company Pages**: Recent hires, team structure, growth
4. **Glassdoor Reviews**: Interview process, company culture, compensation
5. **Crunchbase**: Funding, growth trajectory, key metrics
6. **Stack Overflow Jobs**: Technical requirements, preferred skills

### Key Information to Gather

- **Tech Stack**: Primary languages, frameworks, tools
- **Engineering Culture**: Code review process, deployment practices
- **Recent Projects**: New product launches, technical initiatives
- **Team Structure**: Engineering team size, reporting structure
- **Growth Stage**: Funding round, expansion plans, hiring goals
- **Technical Challenges**: Scale, performance, innovation focus

### Company-Specific Customization Framework

**For each target company, modify**:

1. **Skills Section**: Match their tech stack priorities
2. **Project Descriptions**: Align with their technical challenges
3. **Experience Bullets**: Emphasize relevant scale/domain
4. **Achievement Metrics**: Use comparable scale indicators
5. **Technical Depth**: Match their complexity requirements
